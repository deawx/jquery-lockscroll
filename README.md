# jQuery LockScroll

### A scroll lock plugin

This plugin lock and unlock the scroll of a specific element.

## Usage

### Lock

	$("#element").lockScroll(true);

### Unlock

	$("#element").lockScroll(false);

### Window shortcut

Use $.lockScroll to lock/unlock the window scroll.
